# Our Love Story

A cinematic, romantic frontend experience built with React, Vite, TailwindCSS, Framer Motion, and SwiperJS.

## Run locally

```bash
npm install
npm run dev
```

## Notes

- Replace the placeholder music file at `public/music/romance.mp3` with your own track.
- Memory images use lightweight inline SVGs. Swap them with real photos in `src/data/memories.js` or import from `src/assets/images`.
